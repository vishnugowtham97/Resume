Developed a analog clock using HTML, CSS & Javascript. The HTML code defines the structure of the clock and its hands, while the CSS code styles the clock with a three clock hands representing hours, minutes, and seconds. The JavaScript code updates the rotation of the clock hands every second to reflect the current time, using the setInterval() method to repeatedly call the showTime() function.


The clock hands are positioned and rotated using CSS transforms, and their rotation is calculated based on the current time using the getHours(), getMinutes(), and getSeconds() methods of the Date() object. The hour hand is rotated by an angle that combines the current hour and minute, while the minute hand and second hand are rotated by angles that are based solely on the current minute and second, respectively.
